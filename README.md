# Snapshot Demos

A live demo of **[Snapshot](https://snapshot.djblack.workers.dev)** — publish
interactive Julia to the web, straight from a GitHub repo.

This repo is a small collection of Pluto notebooks. Snapshot compiles each to
run entirely in the browser and hosts them as **one interactive site with a
sidebar**:

**→ [View it live](https://snapshot.djblack.workers.dev/app/Dale-Black/snapshot-demo/snapshot-demo/)**

Snapshot publishes both **notebooks** (a collection like this) and **Therapy web
apps** — connect a repo, push, and it's live in the browser. No server to run.
